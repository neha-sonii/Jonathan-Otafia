# Jonathan Otafia — Personal Brand Website

A modern personal brand website built for Jonathan Otafia, a LinkedIn Visibility Strategist helping coaches and founders turn their expertise into a client acquisition system.

---

## Overview

This website is designed to:
- Showcase Jonathan’s authority and personal brand
- Highlight the 8-Week LinkedIn Visibility System
- Build trust through results, testimonials, and storytelling
- Convert visitors into booked calls

---

## Tech Stack

- React (Vite)
- Tailwind CSS
- Radix UI
- Framer Motion

---

## Features

- Dark + purple brand theme (#7C009E)
- Glassmorphism UI elements
- Scroll-based text reveal animations
- Bento grid layout for client results
- Strong CTA flow (Book a Call)
- Fully responsive design

 ## Performance
